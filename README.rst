EarthReader-TTY
===============

EarthReader client for TTY

Distributed under `GPLv2`__ or later

__ http://www.gnu.org/licenses/gpl-2.0.html


Install
-------

You can install Earth Reader for TTY using ``pip``:

.. code-block:: console

   $ pip install git+git://github.com/earthreader/tty.git
